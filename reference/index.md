# Package index

## Main Functions

Core functions for color classification

- [`charisma()`](https://shawnschwartz.com/charisma/reference/charisma.md)
  : Characterize color classes in biological images
- [`charisma2()`](https://shawnschwartz.com/charisma/reference/charisma2.md)
  : Re-analyze and edit saved charisma objects
- [`color2label()`](https://shawnschwartz.com/charisma/reference/color2label.md)
  : Convert RGB color triplets to discrete color labels

## Validation & Utilities

Tools for validating and working with color classifications

- [`validate()`](https://shawnschwartz.com/charisma/reference/validate.md)
  : Validate Color Look-Up Table completeness
- [`summarize()`](https://shawnschwartz.com/charisma/reference/summarize.md)
  [`summarise()`](https://shawnschwartz.com/charisma/reference/summarize.md)
  : Summarize color classification results

## Visualization

Functions for plotting and displaying results

- [`plot(`*`<charisma>`*`)`](https://shawnschwartz.com/charisma/reference/plot.charisma.md)
  : Plot method for charisma objects
- [`mosaic()`](https://shawnschwartz.com/charisma/reference/mosaic.md) :
  Create a color mosaic visualization from color proportions
